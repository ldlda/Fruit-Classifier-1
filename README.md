# Fruit & Vegetable Classifier

This is a deep learning web application built with Keras and Streamlit.

## Setup

```pwsh
conda create -p <conda folder> python=3
conda activate <conda folder>
echo "`r`n*`r`n" >> <conda folder>/.gitignore # stupid ahh conda
conda install conda-forge::uv
uv pip install -r requirements.txt
uv pip install -r optional-requirements.txt
```
